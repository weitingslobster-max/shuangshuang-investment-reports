CPU 投資機會研究：AMD vs Intel vs Arm

聚焦 CPU 相關投資機會，包含伺服器 CPU、client CPU、hyperscaler 自研 CPU、AI host CPU 與 royalty 模型。非個人化投資建議。

一句話結論：AMD 是目前最適合作為核心持股的 CPU 曝險；Arm 是長期品質最高但估值最貴；Intel 是 turnaround optionality，若已大幅反彈不建議追高。

投資建議

AMD

核心長期 CPU 曝險；適合分批買進，不建議一次追滿

AMD 是目前三者中風險調整後最乾淨的 CPU 投資標的：EPYC 伺服器 CPU 與 Ryzen client CPU 都有成長，且 AI 基礎建設會帶動 host CPU 與平台需求。不過股價與估值已反映不少樂觀預期。

ARM

高品質結構性贏家；適合小部位或等估值回落

Arm 的 royalty 模型很好，受惠於 hyperscaler 自研 CPU、Neoverse、CSS、edge AI 與 physical AI。但目前估值非常昂貴，容錯率低。

INTC

只適合當 turnaround optionality；不建議追高

Intel Q1 2026 顯示營運有改善，DCAI 成長也值得注意；但 foundry 虧損、資本支出、自由現金流、稀釋與政策風險仍高。若股價已大幅反彈，風險報酬不夠乾淨。

建議配置方式

  • 基準配置：若今天要建立 CPU basket：AMD 50%、ARM 25%、INTC 15%、現金 10% 等回檔。
  • 保守配置：AMD 45%、ARM 20%、半導體 ETF / SMH 25%、INTC 0–10%。
  • 為什麼保留現金：CPU 機會是真的，但三家公司都已有高預期或高波動；買點很重要。
買點紀律：CPU 長期機會存在，但好公司不等於任何價格都值得買。AMD/Arm 建議分批；Intel 必須看到 foundry 虧損與現金流改善再加碼。

接下來要追蹤的催化劑

  • AMD：2026/5/5 財報：data-center CPU 成長、MI400/Helios 平台帶動、gross margin 是否維持約 55% 以上、client 庫存健康。
  • Intel：18A/14A 客戶承諾、foundry operating loss 是否收斂、自由現金流、是否繼續稀釋、DCAI 成長能否延續。
  • Arm：2026/5/6 財報：royalty 成長、license lumpiness、CSS 採用率、hyperscaler CPU traction、是否進一步改變商業模式。

主要風險

  • AI capex 可能主要流向 GPU / ASIC，CPU 只拿到較低 margin 的 host attach 價值。
  • 估值風險高：AMD 與 Arm 已經反映強成長；Intel 則提前反映 turnaround 成功。
  • 出口管制、關稅、台海地緣政治、晶圓產能與記憶體供應都可能快速影響毛利。
  • 這是研究輔助，不是個人化投資建議；應搭配部位控制與分批進出。
