CPU Sector 投資研究：AMD、Intel、Arm 與供應鏈定位

聚焦資料中心 CPU、PC CPU、Arm IP、hyperscaler 自研 CPU、AI host CPU，以及 TSMC / NVIDIA / Qualcomm 等延伸標的。這是研究輔助，不是個人化投資建議。

研究流程測試：本報告使用 sub-agent 蒐集產業/供應鏈資料；公司面與風險估值由主流程補完並整合。

一句話結論：CPU sector 不是被 GPU 消滅的夕陽產業，而是從「單純 x86 PC/server」轉成「AI 系統 host CPU + Arm/custom silicon + 先進製程供應鏈」的重組期。若只選直接 CPU 曝險，AMD 是最乾淨的成長標的；Arm 是最好的平台模型但估值容錯低；Intel 是 turnaround / foundry optionality，不是高確定性 compounder。TSMC 則是整個 CPU/AI compute supply chain 的最核心瓶頸資產。

投資結論 / Recommendation

AMD

核心 / core holding

AMD 是最直接、最乾淨的 merchant CPU 成長曝險。EPYC server CPU、Ryzen client CPU 仍在拿市占，且 AI 伺服器也需要 host CPU。不過股價已大幅反映 AI 與 data-center 樂觀預期，適合分批而非一次追滿。

Arm

優先研究 / priority watchlist

Arm 是 CPU 架構轉移的 royalty / IP 平台，能同時受惠 Apple、Qualcomm、NVIDIA Grace/Vera、hyperscaler custom CPUs。但高品質已被高估值反映，適合小部位或等待明顯回檔。

Intel

週期反轉 / cyclical turnaround

Intel 仍有龐大 x86 installed base 與美國本土製造戰略價值，但 foundry 虧損、18A/14A execution、資本支出與現金流是核心風險。若股價已因 turnaround 預期大漲，不宜追高。

TSMC

供應鏈核心 / manufacturing bottleneck

TSMC 不是 CPU 設計公司，但它是 AMD、Apple、Qualcomm、NVIDIA 與大量 Arm/custom CPU 的先進製程瓶頸。若想要較少單一產品風險的 compute infrastructure 曝險，TSMC 仍是核心。

產業定位與供應鏈關聯

  • x86 還沒死：enterprise software compatibility、PC installed base、server fleet inertia 仍支持 Intel/AMD。
  • Arm 正在侵蝕邊界：Apple 已證明 Arm 可以做 premium client computing；AWS/Google/Microsoft/NVIDIA/Qualcomm 則把 Arm 推進 cloud、AI system 與 Windows PC。
  • CPU 角色改變：在 AI data center 裡，CPU 不再是唯一主角，但仍是 host、control plane、storage/networking、inference orchestration 的必要元件。
  • TSMC 是最大共同分母：AMD、Apple、Qualcomm、NVIDIA 與大量 Arm-based custom silicon 都依賴先進製程與封裝；這讓 TSMC 的地位比單一 CPU 公司更穩。

估值紀律 / Valuation discipline

不要把「好趨勢」直接等同「好買點」。 AMD 與 Arm 的品質/成長性較清楚，但估值通常也最貴；Intel 便宜感來自不確定性，不是免費午餐。
  • AMD：若市場已用 AI/data-center 高成長定價，適合用分批、回檔、財報後再評估，而不是在大漲後追價。
  • Arm：可接受 premium multiple，但需要 royalty growth、CSS adoption 與 data-center traction 持續證明；估值過高時只適合小部位。
  • Intel：加碼前應看到 foundry operating loss 收斂、18A/14A 客戶採用、現金流改善，而不只是營收反彈。
  • TSMC：可用作 CPU/AI compute 的基礎持股，但仍需留意地緣政治與 capex cycle。
