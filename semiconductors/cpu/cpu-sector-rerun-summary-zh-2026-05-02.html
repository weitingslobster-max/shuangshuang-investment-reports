一句話結論：CPU sector 已經不是單純的「AMD vs Intel」。比較好的投資框架是五層：x86 merchant CPU、cloud custom Arm CPU、AI host CPU、vertical client silicon、foundry/platform dependency。在這個框架下，TSMC 是風險調整後最穩的 compute infrastructure 核心；AMD 是最直接但估值最敏感的 CPU/AI 成長股；Arm 是高品質 IP tollbooth 但需要嚴格買點；Intel 是 proof-needed turnaround。

投資結論 / Recommendation

TSMC

核心 / core holding

雖然不是 CPU 設計公司，但它是 AMD、Apple、Qualcomm、NVIDIA、hyperscaler Arm/custom silicon 的製造瓶頸。基本面與估值紀律在本組合裡最平衡；主要折價來自地緣政治與出口管制。

AMD

優先研究 / 高成長但高估值

EPYC 與 Ryzen 讓 AMD 是最乾淨的 merchant CPU 成長曝險；Data Center 也受 AI accelerator 帶動。但股價/估值已反映大量 AI upside，現在更適合等財報確認與分批，而非追高。

Arm

平台型優質資產

Arm 是 custom CPU proliferation 的 IP tollbooth，能吃到 AWS、Microsoft、Google、NVIDIA、Qualcomm、Apple 等生態紅利。問題不是品質，而是估值容錯與 royalty growth 是否持續夠快。

Intel

反轉選擇權 / proof-needed

Intel Q1 2026 顯示 DCAI 有成長、現金流有改善，但 foundry 仍大幅虧損。它是 CPU + 美國製造 + foundry turnaround，不是單純便宜的 CPU 股。

Qualcomm

衛星 / value & diversification

Snapdragon X 與 custom silicon 讓 Qualcomm 有 Windows-on-Arm / edge AI optionality；但主體仍是手機與 licensing。估值便宜，角色更像現金回饋與多元化，不是純 AI CPU 押注。

NVIDIA

AI sector weather vane

Grace/Vera 讓 NVIDIA 也在 AI host CPU 佔一席之地；但投資 thesis 仍是 AI accelerator/platform。它的財報與 China commentary 會影響整個 CPU/AI compute basket 的風險偏好。

產業定位與供應鏈關聯

  • x86 仍是現金流核心：enterprise software compatibility 和既有 server/PC fleet 讓 Intel/AMD 仍有強韌性。
  • Arm 的勝利方式不同：它不是靠單一 merchant CPU vendor，而是靠 Apple、Qualcomm、NVIDIA、AWS、Google、Microsoft、Ampere 等客戶的垂直化與客製化。
  • AI 沒有消滅 CPU：AI 反而讓 CPU 角色變成 host orchestration、data movement、inference/control-plane workload 與能源效率優化。
  • foundry 是勝負手：AMD/Apple/Qualcomm/NVIDIA/custom Arm 多數仰賴 TSMC；Intel 的 upside 則來自重建 product/node competitiveness 與 foundry 可信度。

估值紀律 / Valuation discipline

這版最大的差異：我不把 AMD 排在 valuation discipline 第一。AMD 的基本面好，但在 sub-agent risk challenge 裡，它是最需要防高估值壓縮的名字。TSMC 的基本面/估值平衡較好；Qualcomm 較便宜但 CPU 純度低。
  • AMD：若 forward multiple 已把 AI/datacenter 執行打滿，買點要等 5/5 財報、gross margin、MI/EPYC traction 與客戶集中度確認。
  • Intel：不能只因為 turnaround 故事就追高；需要 Foundry operating loss 收斂、外部客戶量產、gross margin 不再卡在 30s。
  • Arm：可接受 premium，但要用 royalty growth、Neoverse/CSS adoption、Q4/FY2026 更新來檢查。
  • TSMC：估值相對合理，但 geopolitical discount 不是噪音，必須靠部位大小和組合分散管理。
  • Qualcomm：估值便宜是優點，但 thesis 是 cash-return + diversification，不是純 CPU/AI upside。
