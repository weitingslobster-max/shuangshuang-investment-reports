4. 半導體供應鏈關聯：它們吃到的是「系統端」需求

  • AI server / data center power：高功率系統需要更多電容、電感、功率 discrete、磁性元件與熱/電源穩定性設計。
  • 車用電動化：EV、ADAS、power electronics、感測器、被動元件與高可靠度規格需求仍是長期驅動。
  • 工業與能源：工廠自動化、renewable energy、inverter、儲能設備會拉動 Vishay/TDK 類零組件。
  • 高頻/RF 與醫療：Knowles 的 RF/microwave filters、高性能電容與 medtech 元件較偏特殊應用，不完全跟 AI server 同步。

所以這一籃不應該被放在 TSM/ASML/AMAT/KLAC 那個「AI 製造瓶頸」層級；更適合放在 AI 與工業電子系統滲透率提高的零組件受益層。

5. 個股看法

TTDKY / TDK：最完整，但不是純被動元件

  • 優點：產品與終端分散，AI server、車用、工業、感測器、磁性元件、能源產品都有曝險。
  • 管理層提到 AI data center/passive component 詢問與訂單上升，但也指出尚未成為主要 earnings driver。
  • 風險：ADR 流動性低、日圓匯率、電池業務波動、Passive Components segment 短期獲利壓力。
  • 我的看法：若想建立「日本電子零組件 + AI server 系統端」曝險，TDK 是這組中最值得先研究與追蹤的。

KN / Knowles：高毛利 niche，但不宜放太大

  • 優點：Q1 gross margin 43.8%、non-GAAP gross margin 45.5%，明顯高於 VSH；產品更 niche。
  • 適合：想要 specialty components、RF、medtech、高可靠度電容曝險的投資人。
  • 風險：規模小、終端集中、流動性較弱；如果成長放緩，估值容易壓縮。
  • 我的看法：可列入 watchlist，但應小部位，等估值/成長能見度更清楚。

VSH / Vishay：分類最準，但品質最週期

  • 優點：公司本身就是 discrete semiconductors + passive electronic components 大廠；4Q book-to-bill 1.20 是復甦訊號。
  • AI-related power applications 被管理層點名，是投資敘事的一部分。
  • 風險：FY2025 gross margin 19.4%、operating margin 1.9%、net loss，顯示議價力與景氣壓力。
  • 我的看法：更像價值/反轉股。除非看到持續 book-to-bill >1、毛利率改善與 EPS 恢復，否則不應當核心。

7. 風險 register

  • 週期性：被動元件、discrete、工業/車用零件容易受庫存週期影響。
  • 低議價力：VSH 類產品毛利率低，競爭與價格壓力明顯。
  • AI 曝險可能被高估：AI server 是亮點，但未必已成為主要 earnings driver。
  • 匯率與 ADR 流動性：TTDKY 是 OTC ADR，交易量低且受日圓影響。
  • 小型股風險：KN 規模小，單一客戶/終端變化可能放大波動。
